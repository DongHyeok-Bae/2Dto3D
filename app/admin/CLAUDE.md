# app/admin - 관리자 대시보드 페이지

**생성일**: 2024-11-22
**Phase**: 2 - 관리자 대시보드

## 📌 목적
프롬프트 관리, 실행 결과 조회, 성능 분석을 위한 관리자 인터페이스

## 📁 페이지 구조

```
app/admin/
├── layout.tsx           # 관리자 레이아웃 (네비게이션)
├── page.tsx             # 대시보드 홈
├── prompts/
│   ├── page.tsx         # 프롬프트 목록
│   └── [phase]/
│       └── page.tsx     # Phase별 편집 페이지
├── results/             # 실행 결과 (예정)
└── analytics/           # 성능 분석 (예정)
```

## 🎨 UI 디자인

### 레이아웃
- 상단 네비게이션: Primary Navy 배경
- 로고 및 메뉴
- 최대 너비: 7xl

### 페이지
- **대시보드 홈**: 빠른 통계 + 카드형 메뉴
- **프롬프트 목록**: 7개 Phase 카드
- **프롬프트 편집**: Monaco Editor + 저장 기능
